.. _whatsnew_040.s_from_xy:

Cumulative distance from GPS coordinates
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Added :meth:`~pandas.DataFrame.pos.s_from_xy` to calculate distance from latitude-longitude.

.. ipython:: python

  df = pd.DataFrame({'lat': [40.0, 40.1, 40.2], 'lon': [-105, -105, -105]})
  df.pos.s_from_xy()
